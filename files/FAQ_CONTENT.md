# FAQ — частые вопросы в левой панели

## Где это в коде
Источник истины — **`frontend/components/FAQSidebar.tsx`**, массив `CATEGORIES`
(каждая категория = `name` + список `{ label, question }`).

- `label` — короткий текст кнопки в панели.
- `question` — полный вопрос, который подставляется в поле ввода чата при клике.

> Раньше данные дублировались в `frontend/lib/faq-data.ts`, но этот файл **не использовался** и удалён. Добавлять/менять вопросы нужно только в `FAQSidebar.tsx`.

## Как добавить вопрос

В `FAQSidebar.tsx` в нужную категорию (или новую) добавить элемент:

```typescript
{
  name: "Депозиты",
  items: [
    { label: "Виды депозитов?", question: "Какие виды депозитов есть в БЦК?" },
    // ...
  ],
},
```

Важно: формулируйте `question` так, чтобы ответ реально был в базе знаний
(`backend/knowledge/`) — иначе ассистент честно ответит «информация не найдена».
