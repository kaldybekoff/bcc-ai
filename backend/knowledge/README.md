# knowledge/ — база знаний BCC

Здесь лежат `.doc` файлы базы знаний (21 файл), разложенные по подпапкам:
`картa/`, `кн/`, `автокредит/`, `депозиты/`, `ипотека/`, `залог/`.

Большинство — MHTML-экспорт из Confluence; один (`картa/общие условия.doc`) — бинарный Word 97.
Формат определяется автоматически в `backend/context.py`:
- MHTML → BeautifulSoup (таблицы → markdown, карточки `aura-card` → `заголовок — значение`, фильтр base64-картинок);
- Word 97 → LibreOffice (`soffice`) → `python-docx`.

`context.py` ищет файлы рекурсивно (`rglob("*.doc")`) — новые файлы и подпапки
подхватываются автоматически при следующем старте сервера.

## Отдельной индексации не требуется

Эмбеддинги для поиска (`retrieval.py`) строятся при старте сервера и кэшируются
в `backend/.embcache/`. При изменении состава базы хэш меняется и эмбеддинги
пересчитываются автоматически. Никаких скриптов индексации запускать не нужно.

Детальный разбор содержимого файлов — в `files/KNOWLEDGE_BASE.md`.
